# WebDAV 可选同步加密实施计划

## 1. 调整后端包格式选择

修改 `src-tauri/src/webdav.rs`：

- 将明文包文件名定义为 `workbuddy-sync.zip`，保留加密包文件名 `workbuddy-sync.zip.enc`。
- 上传时检查同步加密密码。非空时沿用加密流程；为空时直接上传构建出的 ZIP。
- 让公开清单和 latest 清单记录实际上传路径、大小与 SHA-256。
- 移除上传和下载入口中对同步加密密码必填的统一校验。

## 2. 支持自动识别下载格式

继续修改 `src-tauri/src/webdav.rs`：

- 根据清单中的远端包路径判断格式。
- `.zip.enc` 包沿用解密后导入的流程，并在密码为空时返回明确提示。
- `.zip` 包直接作为同步包导入。
- 拒绝未知后缀，避免将任意内容交给导入逻辑。
- 保持已有加密包及其清单结构可用。

## 3. 更新同步设置界面

修改 `src/main.tsx`：

- 在**配置档案**和**同步加密密码**标签旁加入可悬浮、可聚焦的感叹号提示图标。
- 补充两项字段的简短说明。
- 去除同步加密密码的必填限制。
- 根据密码是否为空切换提示文案。
- 密码为空时明确提示会话、模型配置和可能存在的 API Key 将以明文上传为 `workbuddy-sync.zip`。
- 密码非空时说明上传文件为 `workbuddy-sync.zip.enc`。

## 4. 补充提示样式

修改 `src/styles.css`：

- 添加标签与提示图标的对齐样式。
- 添加提示气泡的悬浮和键盘聚焦状态。
- 添加明文上传风险提示样式，并保持现有页面布局和暗色主题一致。

## 完成条件

- 同步加密密码为空时可上传和下载明文 `workbuddy-sync.zip`。
- 同步加密密码非空时继续使用 `workbuddy-sync.zip.enc`。
- 清单始终指向实际上传文件。
- 下载端能自动区分两种格式。
- 旧的加密远端包仍可使用。
- 两个字段均有感叹号悬浮提示，明文模式有清晰风险警告。
